We built NOVA so it could see the actual drawing you're working on.

Every message you send it now attaches a live screenshot of the canvas. NOVA isn't guessing what's there. It's looking at it.

But here's what we fixed today.

Before the fix, if the canvas was black — no drawing loaded — NOVA would still describe what it "saw." Confidently. Like it knew something.

That's the worst version of AI. The kind that sounds right and isn't.

So we added a honesty check. If the canvas is blank or the screenshot fails, NOVA tells you it can't see anything. Simple.

Building AI tools is mostly this. Not the breakthrough moment. The moment you realize it's been lying to you and you fix it.

Real bids deserve real tools.
