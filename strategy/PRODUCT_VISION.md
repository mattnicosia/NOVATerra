## PROGRESSIVE DISCLOSURE (3 Layers)

### Layer 1: The Highway (Every user)
```
DASHBOARD → PROJECTS → [Open Project] → ESTIMATE → REPORTS
```
Five clicks from login to estimate. Primary nav bar.

### Layer 2: Power Tools (Engaged users)
```
PLAN ROOM (AI Scan) → TAKEOFFS → DATABASE → INSIGHTS (3D Model)
```
Secondary tabs inside a project. Discoverable, not demanding.

### Layer 3: Deep Features (Power users)
```
BID LEVELING → ALTERNATES → SPECS → DOCUMENTS → SCHEDULE OF VALUES
```
Collapsible "More" section or contextual access points.

## NAVIGATION RESTRUCTURE

### Global Nav (always visible)
DASHBOARD | PROJECTS | INBOX | PEOPLE | SETTINGS

### Project Tabs (inside a project)
**Primary:** ESTIMATE | PLAN ROOM | TAKEOFFS | REPORTS
**Secondary (collapsible):** Database | Model | Bid Leveling | Specs | More...

### Features to Absorb (not delete)
| Current Page | Better Location |
|---|---|
| Alternates | Section inside Estimate page |
| Specs | Section inside Plan Room |
| Documents | Drawer/panel accessible from any page |
| Schedule of Values | Export format in Reports |
| Bid Leveling | "Compare Bids" action in Projects |
| Intelligence | Merge into Dashboard insights |
| Assemblies | Tab inside Database page |

Result: 20 visible pages → 8-10 primary pages. Same power, simpler surface.
